Do a functional doctest test on the app.
========================================

Let's first create an instance of Sd_buildout at the top level:

    >>> from sd_buildout.app import Sd_buildout
    >>> root = getRootFolder()
    >>> root['app'] = Sd_buildout()

Run tests in the testbrowser
----------------------------

The zope.testbrowser.browser module exposes a Browser class that
simulates a web browser similar to Mozilla Firefox or IE.  We use that
to test how our application behaves in a browser.  For more
information, see http://pypi.python.org/pypi/zope.testbrowser.

Create a browser and visit the instance you just created:

    >>> from zope.app.wsgi.testlayer import Browser
    >>> browser = Browser()
    >>> browser.open('http://localhost/app')

Check some basic information about the page you visit:

    >>> browser.url
    'http://localhost/app'

    >>> print browser.contents
    <html>
    <head>
    <link rel="stylesheet" type="text/css" href="http://localhost/fanstatic/sd_buildout/style.css" />
    <base href="http://localhost/app/@@index" />
    </head>

      this page.</p>

    </body>
    </html>
